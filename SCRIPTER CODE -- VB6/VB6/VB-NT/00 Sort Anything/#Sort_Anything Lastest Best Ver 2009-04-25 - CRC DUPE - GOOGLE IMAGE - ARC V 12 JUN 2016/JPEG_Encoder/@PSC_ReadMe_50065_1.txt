Title: JPEG Encoder Class
Description: Save a picture as a baseline JPG file using only vb code. Open source, NO THIRD PARTY DEPENDENCIES, and as fast as typical commercial software when compiled. All JPEG compression code is contained in a single class that can be added to existing projects. There is also an Integer DCT version of this class written by Ron van Tilburg available on PSC.
