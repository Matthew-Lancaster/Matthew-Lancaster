SUMMARY
=======

Record audio to your hard disk directly in mp3 format.
Many options available, such as only record when sound is present,
record during a certain timeslot,
delete older audio log files when disk space is running low,
or files are older than a certain date,
filesize,
set bitrate, stereo/mono, etc for the resulting mp3 file, etc.
This is freeware, and no support is given. 

For Windows 98/ME


INSTALLATION
============

Unzip the file Mp3Rec.ZIP which contains the following files:

Mp3Rec.exe	executable
README.TXT	this file

Copy these files into a directory of your choice. 
Also add the latest version of lame.exe (http://mitiok.cjb.net/) to the same directory. 
You must also have the VB6 runtimes installed.
You may create shortcuts to the executable, if you wish.

UNINSTALL Mp3Rec
================

Just delete the files and folders that you have added or created, 
and any shortcuts to the executable that you may have created.

USAGE
=====

Run in Windows GUI, adjust settings, and hit Start Write, Auto or Off.
You can run the application on the command line with these options:

Command line options

      /auto  = Enter Auto Mode. Default: Auto off.
      /logx  = logfolder. Default: AppPath\AudioLog\
      /brx   = BitRate x kbits/sec. Default: 8
      /st    = Stereo. Default: Mono
      /durx  = total Duration of Writing x Hours. Default: Forever
      /nfx   = New File every x minutes.Default: 5
      /dp    = Don't Pause on silence. Default: Pause on silence
      /dbx   = x dB limit for pause. Default: -40
      /secx  = x Seconds limit for pause. Default: 10
      /diskx = Min Disk limit x MB. Default: 100
      /sf    = Stop if disk is Full. Default: Delete oldest folder.
      /dk    = Don't Kill old files. Default: Kill old files
      /oldx  = Kill folders Older than x days. Default: 90
