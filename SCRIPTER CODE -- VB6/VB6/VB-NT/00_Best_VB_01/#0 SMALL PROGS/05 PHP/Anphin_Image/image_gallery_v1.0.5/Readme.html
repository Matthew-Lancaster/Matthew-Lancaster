tier lieder's Image Gallery
To Use
Scans directory '/galleries/' relative to gallery.php for directories and builds links to them. When a link is clicked is reads the contents of the specified directory and builds a table of thumbnails linking to the images found. If $cfg['use_thumbnails'] is enabled, the script will look in '/thumbnails/' for an image of the same name and use that for the thumbnail. Additionally if there is a . txt file in '/text/' whose name matches the image selected on the thumbnail table its contents will be read and displayed on the page showing the full sized image.

To Install
Upload contents of "upload" to any drectory on a webserver. That was tough.
