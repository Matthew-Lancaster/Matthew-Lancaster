Title: MS Outlook code -- delete duplicate entries in email, calendar or tasks folder and subfolders
Description: Most of Outlook users have sometimes problems with duplicate entries in email, calendar or tasks folders. This VBA code (3 macros) delete duplicate entries in Outlook folders and subfolders in a safe and speedy way. Each macro give you the possibility to pickup an entry folder. Then, the macro process each entry (delete if match) in the folder and recursivly to all subfolders. In my case, i have delete more then 2000 duplicate emails with no problem.
