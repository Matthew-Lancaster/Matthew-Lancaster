Method [1]: Using Control Panel to modify UAC Settings
CAUTION: Turning off UAC is NOT recommended


---------------------------------------------------------------------------------
From the Start menu, Select Control Panel


---------------------------------------------------------------------------------
Navigate to System and Security > Action Center __ NOT LOOK TRUE
Navigate to System and Security > CHANGE USER ACCOUNT CONTROL SETTINGS __ CORRECT

---------------------------------------------------------------------------------

From the left pane, Select Change User Account Control settings
Drag the scroll button to Never notify
Click on OK to save your settings
