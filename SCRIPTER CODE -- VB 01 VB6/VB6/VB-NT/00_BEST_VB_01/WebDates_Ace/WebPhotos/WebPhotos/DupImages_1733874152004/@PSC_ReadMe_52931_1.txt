Title: DupImages (1.0.1)
Description: DupImages search for bmp/jpg/gif images of identical filesize, in a folder/subfolders, if any found it will also compare image width/height. If possible duplicates found it will generate a html page displaying those images. When you click an image the included vbscript will hide image from display and delete the file from harddisk. Option of displaying all images in folder/subfolders - an easy and fast way to check/delete some image collection. Added option of displaying confirmation on file delete.
